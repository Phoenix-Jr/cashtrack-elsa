# CashTrack React

Application de gestion financière et de caisse, portée en React avec Vite et React Router.

## Installation

```bash
pnpm install
```

## Développement

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Prévisualisation

```bash
pnpm preview
```

## Structure du projet

- `src/pages/` - Pages de l'application
- `src/components/` - Composants réutilisables
- `src/routes/` - Configuration des routes
- `src/stores/` - Store Zustand
- `src/lib/` - Utilitaires et helpers
- `src/types/` - Types TypeScript
